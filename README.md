## Project Description
This project analyzes VLC Media Player as an open-source software. It includes a report and five shell scripts demonstrating Linux concepts.

## Setup Instructions

1. Install WSL and Ubuntu
2. Update system:
   sudo apt update

3. Install VLC:
   sudo apt install vlc -y

4. Give permission to scripts:
   chmod +x *.sh

## How to Run Scripts

- Script 1:
  ./script1.sh

- Script 2:
  ./script2.sh vlc

- Script 3:
  ./script3.sh

- Script 4:
  ./script4.sh /var/log/syslog error

- Script 5:
  ./script5.sh

## Scripts Description

- Script 1: Displays system information
- Script 2: Checks if a package is installed
- Script 3: Shows directory size and permissions
- Script 4: Analyzes log files
- Script 5: Generates a manifesto file
